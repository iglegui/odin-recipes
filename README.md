# odin-recipes

This repository has links(anchor elements) to three different recipes. Each of them has a description, the ingredients and the steps to prepare them. 

I used some basic html tags like h1, p, img, order and unorder lists as well as absolute(links to pages on other websites) and relative links(links to other pages within our own website. 

The coolest thing i learned was the boilerplate which can be used in vs code just by typing the "!" and choosing the first option(for lazy coders!)
